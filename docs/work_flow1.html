Start: Choose a data provider at the top of the page (and a component if available)

Next: View Current Indicators

  • Select each indicator on the left of the page to see the current indicator metadata
    • Is the metadata correct?
      • If so, there’s no need to download last year’s metadata
    • Does the metadata need updating?
      • If so: go to tab 2, “Download Last Year”, to download a csv file of the current metadata

Download Previous: (as needed)

  • Metadata: download and edit a csv file of your current metadata if it needs updating
  • Data: optionally download last year’s files and manually update them, or use them as a template for this year

Upload new:

  • Browse for files to upload
  • Check your files to make sure they can be automatically ingested
